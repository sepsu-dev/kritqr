# KritQR

**KritQR** — Tool sederhana untuk generate QR code dari teks atau URL secara instan. Gratis, cepat, dan mudah digunakan.

## Fitur

- Generate QR code dari teks atau URL
- Download QR code sebagai PNG (resolusi tinggi 512px) dengan nama file custom
- Desain minimalis dan responsif
- Hasil QR code bisa di-scan dengan kamera HP atau aplikasi QR scanner

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Bahasa**: TypeScript
- **CSS**: Tailwind CSS 4
- **QR Code**: react-qr-code + qrcode
- **Font**: Plus Jakarta Sans

## Memulai

### Prasyarat

- Node.js 20+ atau Bun

### Instalasi

```bash
git clone https://github.com/sepsu-dev/kritqr.git
cd kritqr

npm install
# atau
bun install

npm run dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

### Build Produksi

```bash
npm run build
npm start
```
